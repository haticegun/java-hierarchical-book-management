# 📚 Hierarchical Book Management System

A university project developed in Java for the Data Structures course.

The project implements a hierarchical book management system without using Java's built-in collection classes. The hierarchy is created using custom tree and linked list structures.

## Features

- Add new book titles
- Hierarchical chapter structure
- Tree-based navigation
- Console interface
- Custom string helper functions
- Input validation

## Technologies

- Java
- Object-Oriented Programming
- Data Structures
